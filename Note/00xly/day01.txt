命令：
   命令的格式： 命令名称 [选项] [参数]
    [选项] [参数]:都可以省略
   
   clear ：清屏
   history：查看历史信息
   tree -L 1:查看当前路径下1级目录
    pwd :查看文件路径
     ls :查看当前目录下所有文件
     ls -l :查看文件夹所有信息
     ls  -lh:文件大小显示单位的区别
     ls  -a :查看隐藏文件
     man   :帮助手册
     man 1 + 命令 （ls）
     man 2 + 系统调用（open）
     man 3  + 库函数（printf）
     mkdir  : 新建一个目录/文件夹
     mkdir xly/day01 -p  : 可以新建多级目录 
     vi     : 新建一个1.txt /01test.c
     rmdir  : 删除文件夹/目录
     rm  + 单个文件名称   : 删除单个文件
     rm -r + 目录名称  ：删除目录
 cp + 源文件（准备要包括的文件） + 准备要复制到的路径  : 
       cat  + 文件名称:将文件内容打印到屏幕上 

 
    相对路径和绝对路径：
      绝对路径：从根目录的路径
      相对路径：相对于当前目录的路径

   3.进入Linux系统中，cd /mnt/hgfs 就能看到共享文件夹share
